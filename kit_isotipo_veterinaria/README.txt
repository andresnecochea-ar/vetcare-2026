KIT CORREGIDO – ISOTIPO VETERINARIA
===================================

Este paquete fue armado tomando como base la versión elegida por vos:
- cruz violeta
- calendario
- siluetas de perro y gato
- cuadrícula de agenda en el ángulo superior derecho

Se respetó ese diseño como identidad principal.

CONTENIDO
---------
1) svg/master
   - master-color.svg   -> versión principal en violeta #6F2DBD
   - master-black.svg   -> versión todo negro para impresión blanco y negro
   - master-white.svg   -> versión todo blanco para usar sobre fondos oscuros

2) svg/app_icon
   - app-icon-color.svg -> icono de app sobre fondo violeta
   - app-icon-black.svg -> icono de app sobre fondo negro
   - app-icon-white.svg -> icono de app claro

3) png/master
   Las tres versiones master exportadas en tamaños:
   1024, 512, 256, 192, 180, 167, 152, 144, 128, 96, 72, 64, 48, 32, 24 y 16 px.

4) png/app_icon
   Las tres versiones de icono de app en los mismos tamaños.

5) png/previews
   Vistas de control para comprobar la versión blanca sobre fondo oscuro.

6) source
   Incluye la imagen de referencia elegida.

USO RECOMENDADO
---------------
- App / PWA / acceso directo: app-icon-color
- Cabeceras, pantalla de inicio, branding: master-color
- Impresión B/N: master-black
- Fondos oscuros: master-white
